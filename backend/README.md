# How to Run the Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Activate the virtual environment:
   ```bash
   source venv/bin/activate
   ```
3. Run the server:
   ```bash
   python3 manage.py runserver
   ```
4. The API is available at `http://127.0.0.1:8000/api/status/`.
